## Community (Free)

**For developers and small teams**

- Full execution engine -- unlimited steps and concurrency
- All built-in functions (shell, http, git, assert, transform, render, wait)
- Public module registry
- CLI + MCP server for LLM agents
- Skill documents

**$0 forever. No credit card.**

## Pro ($49/user/month)

**For teams that need collaboration and governance**

Everything in Community, plus:
- Private module registry
- Team workflow sharing
- RBAC
- Audit log streaming
- AI agent governance
- orchstep-inspect & orchstep-report

**$49/user/month or $490/year (save 17%)**

## Enterprise (Custom)

**For organizations with compliance requirements**

Everything in Pro, plus:
- SSO / SAML / SCIM
- Compliance certifications
- SLA + priority support
- Custom module policies
